This is a small project to check which movies in an IMDB list are in netflix. It was motivated by the fact that I got bored of checking which movies in my watchlist I could watch. Any list can be specified in the url and after getting the movies I check using the Netflix Roulette API to check if they are in IMDB and display the information. Unfortunately there are some false positives for movies that are in the DVD plan and there are some images missing.
Url examples:
  • http://carmelaacevedo.herokuapp.com/netflix/ls072734994
  • http://carmelaacevedo.herokuapp.com/netflix/ls000426137
